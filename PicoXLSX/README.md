# PicoXLSX Demo

Demo applications for **PicoXLSX**, a lightweight .NET library for creating Microsoft Excel files (XLSX).

## Status

This demo project is planned. Use cases will demonstrate the core features of PicoXLSX, following a similar structure to the [NanoXLSX demos](../NanoXLSX/).
