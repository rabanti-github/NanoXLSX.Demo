# NanoXLSX.Demo
Repository with demo applications for **NanoXLSX**, **PicoXLSX** and **NanoXLSX4j**

## Content

The project contains demo use cases in several sub projects. Each use case is a full running method that either generates 
or reads XLSX files, using the available functions of the three projects:
- [NanoXLSX (.NET 4.5 and .NET Standard)](https://github.com/rabanti-github/NanoXLSX)
- [PicoXLSX (.NET 4.5 and .NET Standard)](https://github.com/rabanti-github/PicoXLSX)
- [NanoXLSX4j (Java >11)](https://github.com/rabanti-github/NanoXLSX4j)
 
## Functional Scope of the Demos

Currently only the **core functionality** is demonstrated. Functions of extended packages or plug-ins are either demonstrated in 
in their own projects or designated sub projects in this repository.
